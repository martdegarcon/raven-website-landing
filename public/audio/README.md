# Добавление музыки

Чтобы добавить музыку из Звездных войн на сайт:

1. Найдите аудио файл с музыкой из Звездных войн (например, главную тему или Imperial March)
2. Переименуйте файл в `music.mp3`
3. Поместите файл в эту папку (`public/audio/`)

Файл должен быть в формате MP3 и называться `music.mp3`.

**Важно:** Убедитесь, что у вас есть права на использование музыки для вашего сайта.
